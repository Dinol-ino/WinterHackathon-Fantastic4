BlockEstate is a blockchain-based real estate platform that enables fractional ownership of property through tokenization.

The system solves three major problems in traditional real estate:

High entry cost

Low liquidity

Lack of transparency

By converting properties into digital tokens, users can buy, sell, and trade property shares securely and transparently without heavy paperwork or intermediaries.

Target users:
Small investors, property owners, and real-estate platforms.


#Features

Property tokenization using blockchain

Fractional ownership & share trading

Secure user authentication

Real-time property data management

IPFS-based document storage

Transparent transaction history

Tech Stack

Frontend: React, Tailwind CSS

Backend: Node.js, Express

Blockchain: Solidity, Ethers.js, MetaMask

Database: Firebase Firestore

Storage: IPFS (Pinata)
